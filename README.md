# Zip Cheat Sheet
```
sudo apt install zip
```

## **Compress File dan Directory**
- **Compress File**
```
zip nama.zip /path/to/file
```

- **Compress Directory**
```
zip -r nama.zip /path/to/file
```

- **Compress Directory dengan Password**
```
zip -r -e nama.zip /path/to/file
```

## **Ekstrak Archive**
```
unzip nama.zip
```
